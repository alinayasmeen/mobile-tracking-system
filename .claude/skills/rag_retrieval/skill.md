# **Name:** `rag_retrieval`

Provides the agent with retrieval-augmented generation features.

## Capabilities

* Build RAG pipelines (loader → chunker → embedder → Qdrant)
* Create document loaders for PDF, text, HTML
* Generate embeddings and store in Qdrant
* Write retrieval endpoints in FastAPI
* Integrate with OpenAI embeddings
* Create RAG-ready prompt templates
